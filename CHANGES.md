- [`3afb659`](https://github.com/sinonjs/referee-sinon/commit/3afb6597985256f69e6b02d4b4b6d5a69e05dba9)
  Refactor callsToString: omit extra line breaks (Morgan Roderick)
    >
    > This version has a couple of interesting changes:
    >
    > * it does not output extra line breaks (see below)
    > * it has 100% coverage with existing tests
    >
    > ```js
    > var spy = sinon.spy();
    > spy("hello\nworld");
    > spy("hello world");
    > spy("hello world");
    > spy("hello world");
    >
    > try {
    >     refute.called(spy);
    >     throw new Error("Exception expected");
    > } catch (e) {
    >     var message =
    >         "[refute.called] Expected function spy() " +
    >         "{} to not be called but was called twice";
    >     assert.equals(e.message, message);
    > }
    > ```
    >
    > \#### Before
    >
    > ```
    > -[refute.called] Expected function spy() {} to not be called but was called 4 times
    > -    spy(hello
    > -world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:405:21)
    > -
    > -    spy(hello world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:406:21)
    > -
    > -    spy(hello world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:407:21)
    > -
    > -    spy(hello world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:408:21)
    > +[refute.called] Expected function spy() {} to not be called but was called twice
    >
    > ```
    >
    > \#### After
    >
    > ```
    > -[refute.called] Expected function spy() {} to not be called but was called 4 times
    > -    spy(hello
    > -world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:405:21)
    > -    spy(hello world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:406:21)
    > -    spy(hello world) at Context.<anonymous> (/Users/morgan/code/sinonjs/referee-sinon/lib/referee-sinon.test.js:407:21)
